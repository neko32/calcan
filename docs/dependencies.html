calcan 依存ライブラリ一覧 🐾

線形代数計算ドリルアプリ「calcan」で使用しているプロダクションおよび開発用の主要な依存パッケージ一覧まる。

ライブラリ名 タイプ 目的 / 役割
react Production ユーザーインターフェースを構築するためのコアライブラリ。コンポーネントツリーと状態管理を担当。
react-dom Production ReactをブラウザのDOMにマウントし、Web上で動作させるためのエントリポイント。
katex Production 数式を高速かつ美しくブラウザ上にレンダリングするためのデファクトスタンダードライブラリ。
lucide-react Production 美しくシンプルなSVGアイコンを提供するアイコンパックライブラリ。
typescript Development 型安全性と高度なコード補完を開発時にもたらす静的型付け言語コンパイラ。
vite Development 高速なホットリロード (HMR) とビルド最適化を行う次世代ビルド・開発ツール。
vitest Development Viteと緊密に統合され、非常に高速に動作するテストランナー・フレームワーク。
@testing-library/react Development Reactコンポーネントをユーザーの操作視点で検証するためのUIテスト用ライブラリ。
jsdom Development Node.js環境下でブラウザの挙動をシミュレートするヘッドレスDOM実装（テスト実行時に使用）。
@vitest/coverage-v8 Development テスト実行時のコードカバレッジを詳細に測定・分析しHTML等のレポートを生成するツール。
